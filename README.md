# CMC

目的：缓解地球脆弱、过度负荷的生态系统压力
对火箭的要求： routine, safe, and inexpensive



# 月球殖民地运输建模信息解构

## 1. 未来设想背景
- **目标**：让任何人都能轻松、安全、低成本进入太空，建造可居住、度假或工作的太空栖息地。
- **潜在效益**：
  - 缓解地球生态压力
  - 提供环保、高效、全球化的太空运输基础设施
- **技术路径**：电力驱动的**空间电梯系统（Space Elevator System）**，作为可扩展的行星际物流和探索基础设施。

## 2. 空间电梯系统配置
- **最终结构**：
  - 3 个银河港（Galactic Harbours），地球赤道上间隔 120°
- **银河港组成**：
  - 1 个地球港（Earth Port）
  - 2 个天顶锚点（Apex Anchors）
  - 2 根缆索（Tethers），长度 100,000 km
  - 多台电梯协同工作
- **功能**：
  - 日常运输大量货物从地球 → 地球同步轨道（GEO） → 天顶锚点
  - 天顶锚点 → 火箭 → 月球/其他星球
  - 电梯运输阶段无污染
- **银河港运力**：
  - 每个港口每年运输 179,000 吨
  - 运输为两步走：
    1. 电梯：地球港 → Apex Anchor
    2. 火箭：Apex Anchor → 月球殖民地

## 3. 月球殖民地需求
- **目标人口**：100,000 人
- **总建设物资**：100,000,000 吨（1 亿吨）
- **后期日常供应**：
  - 水和其他生活物资需定期运输
- **建设启动年份**：2050（在空间电梯完成后开始）

## 4. 火箭运输方案
- **发射基地（10 个）**：
  - 美国：阿拉斯加、加州、德州、佛州、弗吉尼亚
  - 国际：哈萨克斯坦、法属圭亚那、印度 Satish Dhawan、 中国太原、 新西兰 Mahia Peninsula
- **运力**：
  - 每次发射：100–150 吨（平均 125 吨）
  - 火箭运输为一步完成：地球 → 月球
- **假设**：
  - 2050 年火箭先进，假设完美发射（无失败）

## 5. 建模任务要点
- **核心目标**：建立数学模型计算：
  1. **成本**（Cost）
  2. **运输时间/建设时间**（Timeline）
- **比较对象**：
  - 3 个银河港的空间电梯系统
  - 传统火箭从选定基地发射
- **情景要求**：
  - 纯电梯方案
  - 纯火箭方案
  - 混合方案（电梯 + 火箭）
- **可假设条件**：
  - 银河港系统与火箭运输条件理想（无失败、无缆索摆动）
- **延展分析**：
  - 系统不完美（电梯故障、火箭发射失败）对时间与成本影响
  - 后续供应（如水）需求及影响
  - 环境影响对比与优化

## 6. 核心参数汇总

| 参数 | 数值/说明 |
|------|-----------|
| 银河港数量 | 3 |
| 电梯年运输能力 | 179,000 吨 / 港口 |
| 缆索长度 | 100,000 km |
| Apex → 月球火箭 | 电梯输送至 Apex，再用火箭运输 |
| 总物资 | 100,000,000 吨 |
| 月球人口 | 100,000 人 |
| 火箭载荷 | 100–150 吨 / 发（平均 125 吨） |
| 火箭基地 | 10 个（全球分布） |
| 任务开始年份 | 2050 |
| 假设 | 系统完美运行（初始模型） |

## 7. 可建模思路提炼
1. **运输能力模型**：
   - 电梯系统：年运力 × 港口数量 → 总年运力
   - 火箭系统：发射次数 × 单次载荷 → 总年运力
2. **时间估算**：
   - 完工时间 = 总物资 ÷ 年运力
   - 混合方案时间 = 并行运力中耗时最长者
3. **成本模型（参数化）**：
   - 电梯成本 = 单吨成本 × 运输吨数
   - 火箭成本 = 单发成本 × 发射次数
4. **敏感性分析**：
   - 考虑失败率或停运 → 有效运力下降
   - 考虑后续水、生活物资补给
5. **环境影响分析**：
   - 电梯阶段 ≈ 无排放
   - 火箭阶段 → 二氧化碳或其他污染


# 术语表

1. **空间电梯系统（Space Elevator System）**  
   - 由三个**银河港（Galactic Harbours）**及其他辅助设施组成。  
   - 用于将物资从地球表面运输到太空，实现高效、环保的太空物流。

2. **银河港（Galactic Harbour）**  
   - 每个银河港由两个**天顶锚点（Apex Anchors）**组成，每个锚点通过两根**缆索（Tethers）**与一个地球港口（Earth Port）连接。  
   - 类似太空中的物流枢纽，是空间电梯运输的起点与终点。

3. **地球港口（Earth Port）**  
   - 位于地球上的设施，为银河港提供地面支撑。  
   - 功能包括物资装载、设备维护、能源供应等，相当于“太空码头”。

4. **缆索（Tethers）**  
   - 长度约 **100,000 公里**，使用**石墨烯（graphene）**制造。  
   - 连接地球港口与天顶锚点，承载电梯升降物资，是空间电梯的关键结构。

5. **天顶锚点（Apex Anchor）**  
   - 位于太空中的配重，用于维持缆索张力和系统稳定。  
   - 同时作为物资从电梯转运到火箭的中转点。

6. **地球同步轨道（Geosynchronous Orbit, GEO）**  
   - 高度约 **35,786 公里**，绕地球一圈的时间为 **24 小时**，与地球自转周期一致。  
   - 轨道上的物体会固定在地球某一经度上空，适合用于通信卫星或空间电梯中转。

7. **月球殖民地（Moon Colony）**  
   - 建在月球上的居住区，可容纳 **100,000 人**。  
   - 目标是长期居住、工作和生活，支持大规模人类活动。
